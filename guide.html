⚡ 4 bước bắt đầu

  1. Cài extension vào Chrome (Load unpacked)
  2. Lấy API Key từ OpenAI Platform
  3. Dán key → Test kết nối → Lưu
  4. Nhấn Alt + Q → Kéo chọn vùng → Xong!
1

📦 Cài Extension vào Chrome

  1. Mở Chrome
  2. Gõ vào thanh địa chỉ:
    chrome://extensions/ 📋
  3. Bật Developer mode (góc trên bên phải)
  4. Click "Load unpacked"
  5. Chọn thư mục ai-translate-extension
💡 Click 🧩 (Extensions) → Ghim 📌 "AI Screen Translator" lên toolbar
2

🔑 Lấy API Key OpenAI

  1. Truy cập:
    https://platform.openai.com/api-keys 🔗
  2. Đăng nhập tài khoản OpenAI
  3. Click "Create new secret key"
  4. Copy API Key — dạng: sk-...
⚠️ Bảo mật: API Key là riêng tư. Key được lưu cục bộ trong Chrome. Cần có credit trong tài khoản để sử dụng.
3

⚙️ Cấu hình Extension

  1. Click icon extension 🌐 trên toolbar
  2. Dán API Key vào ô "OpenAI API Key"
  3. Click 🧪 Test kết nối — kiểm tra key hoạt động MỚI
  4. Chọn Model AI:
    ModelƯu điểmGiá
    GPT-4oChất lượng tốt nhất~$2.50/1M tokens
    GPT-4o-miniNhanh, rẻ, đủ dùng~$0.15/1M tokens
    GPT-4.1Model mới nhất~$2.00/1M tokens
    GPT-4.1-miniCân bằng giá/chất lượng~$0.40/1M tokens
  5. Chọn ngôn ngữ dịch → Click 💾 Lưu cài đặt
💡 Chỉ cần cấu hình 1 lần. Extension nhớ cài đặt vĩnh viễn.
4

🎯 Cách sử dụng

Cách 1: Phím tắt

  1. Nhấn Alt + Q
  2. Màn hình tối đi, con trỏ thành dấu +
  3. Kéo chuột khoanh vùng text cần dịch
  4. Thả chuột → Text stream real-time trong panel bên phải ⚡

Cách 2: Chuột phải MỚI

  1. Chuột phải trên trang bất kỳ
  2. Chọn 🌐 Chụp & Dịch vùng này

Chế độ OCR MỚI

  1. Mở popup → chọn 📝 Chỉ trích text (OCR)
  2. Chụp vùng → AI trả về text gốc chính xác, không dịch
⚡ Streaming: Text hiện từng chữ real-time ngay khi AI xử lý, không cần đợi hết mới thấy kết quả!
📋 Mỗi đoạn dịch có nút 📋 Copy riêng. Hoặc copy toàn bộ, lưu file .txt, xóa lịch sử từ thanh công cụ panel.
